Classification Category

Each occurrence of Classification Category defines and name one well-defined grouping within a Classification Set for analyzing Legal Entities.

Attributes

The attributes show relationships between the data group and none-group data nodes, these are data nodes that carry individual pieces of information

Name Type Documentation Sample Value Available Sources
ClassificationCategoryName String The Name for this Category within this Classification Set. AUTOMOTIVE
ClassificationCategoryCode Enumeration A standard abbreviation for this Classification within this Classfication Set. AUTO
ClassificationSequenceNumber Number A number that allows the Category members of a Classification Set to be arranged in a defined order. 3.0

Relationships

The group relationships show relationships between this group and other data groups in the schema.
