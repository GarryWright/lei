LE Document

Each occurrence of LE Document references one piece of written, printed, or electronic matter pertaining to one Legal Entity. The Document may be stored in any number of different file formats (PDF, Word, Text, HTML, etc.) as indicated by the Document File Type attribute. The actual Document contents is provided via the Document Link ID attribute.

Attributes

The attributes show relationships between the data group and none-group data nodes, these are data nodes that carry individual pieces of information

Name Type Documentation Sample Value Available Sources
DocumentBriefDescription String A short synopsis of the Document contents. Corporate Bylaws
DocumentDate Date The date the Document was published. 15-May-1992
DocumentAuthor String The name of the Author of the Document UNKNOWN
DocumentLinkID String The file ID, URL or similar link to one LE Document. HTTP://WWW.CORP.COM/DOC/0515199205
DocumentFileType Enumeration A code that indicates the file or technical format in which the Document is stored. PDF
