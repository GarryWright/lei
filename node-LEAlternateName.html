LE Alternate Name

Each occurrence of LE Alternate Name provides an additional name or alias in a designated language provided by a Data Source by which a Legal Entity is known. Every LE has one Legal Entity Name attribute that is stored on the Legal Entity and may have "n" number of additional alternate names.

Attributes

The attributes show relationships between the data group and none-group data nodes, these are data nodes that carry individual pieces of information

Name Type Documentation Sample Value Available Sources
AlternateNameType String The name or term used for the entire class of this particular type or genre of Alternative Name 1 SHORT NAME
LEAlternateNameImage String A picture or image of a Legal Entity Alternate Name, for cases where the text character set for the Language of the Name is not supported. "--"
LanguageofNameCode Enumeration A code indicating the character set of the language used for an Alternate Legal Entity Name. ENGLISH
LEAltNameStartDate Date The first date on which an Alternate Name was first in effect. 15-Jan-2005
LEAltNameEndDate Date The last date on which an Alternate Name is in effect. "--"
LEAltNameStatus Enumeration The state or status of this name regardings it's use. ACTIVE
