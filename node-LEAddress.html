LE Address

Each occurrence of Legal Entity Address provides one Address for one Legal Entity for a designated period of time. Each Legal Entity may have any number of Addresses which may be further qualified by the Address Role attribute. The same Address may be used for any number of Individuals and/or Legal Entities. The Individual Address Start and End Date in conjunction with the Cycle Frequency attribute allows an address to be used for a specified period of time and then be repeated on a designated cycle, such as for a summer to be used during the summer months only on a yearly basis.
