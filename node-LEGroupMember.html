LE Group Member

Each occurrence of LE Group Member defines one Legal Entity that is a member of one LE Group. Each individual Legal Entity may belong to any number of different LE Groups. Conversely, each LE Group may have any number of Legal Entities as its members.

Attributes

The attributes show relationships between the data group and none-group data nodes, these are data nodes that carry individual pieces of information

Name Type Documentation Sample Value Available Sources
MembershipStartDate Date The earliest or beginning of period date in which a LE was first a member of a LE Group. 15-Feb-2012
MembershipEndDate Date The latest or end of period date in which a LE was last a member of a LE Group. "--"
